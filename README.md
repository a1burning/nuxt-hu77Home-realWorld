# Fox Home
## 介绍
这个项目是RealWorld为模板，用nuxtJS为框架写的完整示例，实例中包括：

- 登录、注册、登出
- 文章列表、用户关注的人文章列表、tag文章列表
- 个人中心、编辑个人信息
- 个人文章列表、点赞文章列表
- 发表文章、编辑文章、删除文章
- 文章下发表评论、删除评论
- 点赞、取消点赞
- 关注、取消关注
- ...
